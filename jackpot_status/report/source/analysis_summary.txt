# 밈코인 시장의 잭팟 추구형 투자자 분석 결과 요약

## 데이터
- 분석 대상: 7,643개 지갑의 거래 행동 데이터
- 분석 기간: 데이터에 포함된 거래 기간
- 분석 지표: 10개 지표 (EXPECTED_ROI, ROI_STANDARD_DEVIATION, SHARPE_RATIO 등)

## 주요 발견사항
1. 밈코인 투자자들은 크게 2개의 군집으로 분류됨
   - 군집 1 (잭팟 추구형): 높은 ROI 범위, 높은 변동성, 높은 기대 수익률
   - 군집 0 (일반 투자자): 상대적으로 낮은 변동성과 수익률 범위

2. 잭팟 추구형 투자자 식별에 중요한 4개 핵심 지표(중요도 순):
   - ROI_RANGE: 수익률 범위
   - ROI_STANDARD_DEVIATION: 수익률 표준편차
   - EXPECTED_ROI: 기대 수익률
   - SHARPE_RATIO: 샤프 비율

3. 두 가지 식별 방법 개발:
   - 룰 기반: 각 핵심 지표에 대한 임계값 설정 (OR 조건)
   - 스코어 기반: 가중치를 적용한 종합 점수 계산

4. 두 방식 간 일치율: 약 78.5%

## 실무 활용 방안
1. 투자자 세그먼트 관리
2. 위험 관리 및 시장 변동성 예측
3. 타겟 마케팅 전략 수립

## 향후 연구 방향
1. 시간에 따른 투자 행태 변화 분석
2. 잭팟 추구형 투자자의 구체적 거래 전략 심층 분석
3. 예측 모델 개발
